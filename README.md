Requirements
----
<command> (<apt package>)

The following libraries are necessray to build the "pwave" program:
- gsl (libgsl0-dev)

In order to run the "mkvideo" program, make sure binaries for the following are in your PATH:
- h5import (hdf5-tools)
- h5topng (h5utils)
- avconv (libav-tools)
- convert (imagemagick)
